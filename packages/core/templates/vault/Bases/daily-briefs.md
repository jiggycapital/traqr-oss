# Daily Briefs

Bases view for all daily briefs generated by the /daily-brief skill.

> **Setup:** When Obsidian Bases plugin is installed, convert this to a `.base` file with:
> - View: **table**
> - Source: `00 Inbox/Briefs/`
> - Sort by: `created` (descending)
> - Columns: title, created, domain, status
> - Filter: `type = brief`

# Research Tracker

Bases view for all research documents across projects.

> **Setup:** When Obsidian Bases plugin is installed, convert this to a `.base` file with:
> - View: **table**
> - Source: vault root
> - Sort by: `created` (descending)
> - Group by: `domain`
> - Columns: title, domain, status, created, source
> - Filter: `type = research`

## Research Domains

- `pokotraqr` — PokoTraqr companion app
- `nooktraqr` — NookTraqr island management
- `platform` — Traqr platform infrastructure
- `life` — Personal/Life OS features
- `jiggy` — Jiggy Capital investment tools
